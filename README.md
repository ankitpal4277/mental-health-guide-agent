# Mental Health Guide Agent

A compassionate AI-powered mental health check-in agent built using Google ADK and Gemini.

## What it does
- Greets the user warmly
- Listens to how the user is feeling
- Saves the user's mood to state
- Searches Wikipedia for relevant mental health coping strategies
- Responds with empathy, helpful tips, and a reminder to seek professional help

## Tech Stack
- Google ADK
- Gemini 2.5 Flash
- LangChain Wikipedia Tool
- Google Cloud Run

## Live Demo
https://mental-health-guide-620139818570.us-central1.run.app

## Setup
1. Clone the repository
2. Copy .env.example to .env and fill in your values
3. Install dependencies: pip install -r requirements.txt
4. Run locally: adk run mental_health_guide_agent
